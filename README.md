# Topics for today

## Git

* git add <file>
	add file to staging area
* git commit -m "Message"
	put files to local repo
* git push
	push all commits to repo
* git restore <file>
	take back your modification, but only before commit
* git rm <file>
	remove file from repo and local
* git rm --cached <file>
	remove file from cache
